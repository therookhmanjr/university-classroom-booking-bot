# University Classroom Booking Bot

## Описание проекта
Система представляет собой Telegram-бота, предназначенного для просмотра расписания занятий и бронирования учебных аудиторий университета.
Пользователи могут просматривать расписание, находить свободные аудитории и создавать заявки на бронирование. Администратор управляет пользователями, расписанием, аудиториями и рассматривает поступившие заявки.

## Возможности
Пользователь: 
* Просмотр расписания
* Поиск свободных аудиторий
* Бронирование аудитории
* Просмотр своих заявок

Администратор:
* управление пользователями
* управление аудиториями
* управление расписанием
* подтверждение заявок
* отклонение заявок

## Стек технологий
Backend: Python  
Telegram Framework: aiogram 3  
База данных: Microsoft SQL Server  
Работа с БД: pyodbc  
Дополнительные библиотеки: python-dotenv  

## Архитектура
Проект реализован по клиент-серверной архитектуре.  

В качестве клиентского приложения используется Telegram, через который пользователь взаимодействует с системой. Все команды и действия пользователя передаются через Telegram Bot API в серверную часть приложения.  

Серверная часть разработана на языке Python с использованием библиотеки Aiogram. Она обрабатывает входящие сообщения, проверяет права доступа пользователя, выполняет бизнес-логику приложения и взаимодействует с базой данных.

Для хранения информации используется Microsoft SQL Server. В базе данных содержатся сведения о пользователях, корпусах, аудиториях, расписании занятий и заявках на бронирование.

Обмен данными между серверной частью и базой данных осуществляется посредством SQL-запросов через библиотеку pyodbc.
